# Covid 19

Covid19 is an example app that show how to use:

* Combine
* SwiftUI

## Installation

The dependencies are listed in the Podfile

```bash
pod install 
```

## Usage

The documentation of the API that this app uses are in this link [CovidAPI](https://disease.sh/docs/#/)
